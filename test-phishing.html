How to Test

  1. Make sure the QuickPhish extension is installed and enabled
  2. Open Chrome DevTools (F12) and go to the Console tab
  3. Click on each link in the test groups
  4. Watch the console for API responses and detection logs
  5. For safe links, you should be redirected normally
  6. For phishing links, you should see the warning modal
  7. For special links (mailto:, javascript:), they should work normally
Debugging Tips:
  • If you don't see the warning modal, check the console for any errors
  • Make sure the extension is properly loaded in chrome://extensions/
  • Verify that the content script is running by looking for initialization logs in the console
